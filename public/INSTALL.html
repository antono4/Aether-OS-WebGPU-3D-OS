Installation Guide

Get started with Aether OS in minutes

📋 Prerequisites

Before installing Aether OS, ensure you have the following:

18+
Node.js
9+
npm
2.0+
Git
Chrome 94+
Browser

🚀 Quick Start (Web)

# Clone the repository
git clone https://github.com/antono4/Aether-OS.git
cd Aether-OS/spatial-canvas

# Install dependencies
npm install

# Start development server
npm run dev

Open http://localhost:5173 in your browser.

🐳 Docker (Production)

# Build and run with Docker
docker build -t aether-os .
docker run -p 5173:80 aether-os

Or use docker-compose:

docker-compose up -d

🤖 AI Setup (Optional)

For AI features, create a .env file:

CLAUDE_API_KEY=your_claude_key
DEFAULT_AI_PROVIDER=claude

Or use local Ollama (no API key needed):

ollama pull llama3
ollama serve
Launch Aether OS →

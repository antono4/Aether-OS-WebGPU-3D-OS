API Documentation

Complete API reference for building with Aether OS

GET Memory Nodes

Retrieve nodes from the associative memory graph

memoryGraph.getNode(id: string): MemoryNode

POST Create Node

Create a new node in the memory graph

memoryGraph.createNode(config: NodeConfig): MemoryNode

PUT Update Node

Update an existing memory node

memoryGraph.updateNode(id: string, data: Partial<NodeData>)

DELETE Delete Node

Remove a node from the graph

memoryGraph.deleteNode(id: string): void

GET Search

Semantic search across all nodes

memoryGraph.searchNodes(query: string, limit?: number)

POST AI Complete

Generate AI completion

ai.complete(prompt: string, options?: AIOptions)

POST Event Emit

Emit custom events through the event bus

eventBus.emit(event: string, data?: any): void

GET Subscribe

Subscribe to event bus events

eventBus.on(event: string, callback: Function): Unsubscribe

Quick Start Example

import { AetherOS } from './main'; const aether = new AetherOS(); // Create a memory node const node = aether.memoryGraph.createNode({ type: 'text', content: 'My brilliant idea', tags: ['creative', 'project'], importance: 0.9 }); // Search semantically const results = await aether.memoryGraph.searchNodes('ideas about projects'); // Subscribe to events aether.eventBus.on('node:created', (data) => { console.log('Node created:', data.id); });
